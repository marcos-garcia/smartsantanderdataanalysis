Smart Santander Data Analysis

Proyecto de tratamiento de datos de la SmartCity de Santander dentro de una arquitectura lambda, mediante el uso de herramientas Big Data.

Ver en GitHub

Recolección y almacenamiento

A partir de los datos que emiten los sensores de la Smart City, y que se publican en una web, se genera un almacén de datos distribuido desde el cual se pueden hacer diversos análisis de la información y visualizaciones.

Descripción de la Arquitectura del Sistema

Análisis y visualización

Los datos se agregan mediante jobs de Apache Spark y se almacenan en una base de datos Cassandra para su posterior visualización web.

Empleo de la librería MLLib de Machine Learning para utilizar el algoritmo de clustering en la clasificación de días según ciertas medidas recogidas y visualización de sus resultados en la web.

Introducción

En este proyecto se ha desarrollado un sistema de extracción de datos de sensores colocados en la ciudad de Santander que se publican a través de una interfaz web para un posterior tratamiento utilizando tecnologías Big Data.

Como resultado se ha construido un almacén de datos distribuido con las mediciones realizadas durante el tiempo de extracción de información, así como varias interfaces web para la visualización de estas en datos agregados que se generan tanto desde un proceso batch como en otro real-time utilizando el patrón de arquitecturas lambda.

También se ha desarrollado un ejemplo de tratamiento analítico de la información recogida utilizando la técnica de clustering para clasificar los días de recogida de información en varios grupos, según la evolución de alguna de las medidas leídas a lo largo de las horas.
